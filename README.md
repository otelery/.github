# .github

This repository holds the default community health files and GitHub organization profile for [Otelery](https://github.com/otelery).

## What lives here

| Path | Purpose |
|------|---------|
| `profile/README.md` | Organization profile shown on the Otelery GitHub page |
| `CODE_OF_CONDUCT.md` | Community standards for all Otelery repositories |
| `CONTRIBUTING.md` | Guidelines for contributing to any Otelery project |
| `SECURITY.md` | How to report security vulnerabilities |
| `SUPPORT.md` | Where to get help |

Files placed here are automatically used as defaults across all repositories in the Otelery organization that do not define their own.
